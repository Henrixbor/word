# Word! Asset Brief

## Brand Direction
- Working title: `Word!`
- Visual inspiration: broad-market social word game, high-energy, toy-like, glossy, welcoming
- Loading screen mood: bold, celebratory, almost board-game packaging energy
- Character direction: cute smiling dachshund, expressive eyes, slightly oversized head, soft rounded proportions
- Rendering style: premium AI-rendered 3D illustration, polished plastic/felt hybrid surfaces, not flat vector
- Background handling: transparent PNG character assets; separate full-bleed loading backgrounds

## Mascot Rules
- Breed must read instantly as dachshund
- Smile must feel friendly, not uncanny
- Long body, short legs, floppy ears
- Warm caramel / chestnut palette with cream muzzle accents
- Keep silhouette clean enough for icon crops
- No collar tags with text
- No photorealism
- No line-art/vector look

## Prompt Principles
- Use the exact same mascot identity across all generations
- Prefer a single seed/reference workflow after the first approved mascot render
- Keep lighting soft but high-contrast enough for mobile thumbnails
- Avoid clutter near edges for icon/splash crops
- Favor transparent backgrounds for character-only assets

## Approval Sequence
1. Approve mascot identity first
2. Then generate logo/icon system around the approved mascot
3. Then generate loading/onboarding illustrations in the same style
